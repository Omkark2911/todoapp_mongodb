# todoapp_mongodb
This project is a simple Todo Backend API built using Node.js, Express, MongoDB, and JWT for authentication. It allows users to sign up and sign in securely, after which they receive a token that must be used to access protected routes. Each authenticated user can create, view, and delete their own todos, ensuring proper data isolation using a userId field stored with every todo. The backend uses Mongoose to interact with MongoDB and handles core CRUD operations for managing tasks. While building this project, key concepts like REST APIs, middleware, authentication using JWT, and database integration were implemented and understood.
